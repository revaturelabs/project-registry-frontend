Instructions

How to:

  • Filter Projects
    • You can filter by project tags, phase, name, status. To reset the project view, press Reset Filters button
  • Create New Project
    • Click the Create Project button. Add a name and description in the Create Project Page.
  • Update Project
    • Click the row of the project you wish to update to be redirected to the update project page.
  • Create Iteration
    • To create a new iteration of a project, select a batch from the drop down iteration box and click the new iteration button.
